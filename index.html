1. Pengenalan C++

1.1 Definisi dan Sejarah

Apa Itu Bahasa Pemrograman C++?

C++ adalah bahasa pemrograman tingkat tinggi yang dikembangkan oleh Bjarne Stroustrup pada awal tahun 1980-an di Bell Laboratories. C++ adalah pengembangan dari bahasa pemrograman C, dengan tambahan fitur-fitur baru seperti paradigma pemrograman berorientasi objek (OOP).

Keunggulan C++:

  • Fleksibilitas yang tinggi
  • Kemampuan menghasilkan kode yang efisien
  • Dukungan untuk pengembangan sistem operasi, game, dan aplikasi embedded
  • Paradigma pemrograman berorientasi objek

Sejarah dan Evolusi C++

Tahun Versi Fitur Utama
1983 C++ Awal Ekstensi dari bahasa C
1998 C++98/C++03 Standardisasi ISO pertama
2011 C++11 Manajemen memori, fitur OOP, pemrograman generik
2020 C++20 Modul, konsep pemrograman fungsional

2. Variabel dan Tipe Data

2.1 Pengenalan Variabel

Variabel adalah suatu lokasi di memori yang diberi nama dan digunakan untuk menyimpan data. Variabel memiliki tipe data yang menentukan jenis data yang dapat disimpan di dalamnya.

Deklarasi Variabel

int angka; // Deklarasi double pi; // Deklarasi

Inisialisasi Variabel

int angka = 10; // Inisialisasi saat deklarasi double pi = 3.14159; // Inisialisasi saat deklarasi // Inisialisasi terpisah int x; x = 20;

5. Perulangan

5.1 For Loop

For loop digunakan untuk iterasi dengan jumlah perulangan yang sudah diketahui.

Struktur For Loop

for (inisialisasi; kondisi; increment/decrement) { // Kode yang akan diulang }

Contoh For Loop Ascending (1, 2, 3, 4, 5)

#include using namespace std; int main() { // Mencetak angka 1-5 for (int i = 1; i <= 5; ++i) { cout << i << " "; } cout << endl; return 0; }

Contoh For Loop Descending (5, 4, 3, 2, 1)

#include using namespace std; int main() { // Mencetak angka 5-1 for (int i = 5; i >= 1; --i) { cout << i << " "; } cout << endl; return 0; }

Latihan For Loop dari 1 sampai 100 Ascending

#include using namespace std; int main() { // Ascending (1 sampai 100) cout << "Ascending (1 ke 100):" << endl; for (int i = 1; i <= 100; i++) { cout << i << " "; } return 0; }

10. Algoritma Sorting & Searching

10.1 Algoritma Sorting

1. Bubble Sort

Bubble Sort membandingkan elemen yang berdekatan dan menukar posisinya jika elemen pertama lebih besar dari elemen kedua.

void bubbleSort(int arr[], int n) { for (int i = 0; i < n-1; i++) { for (int j = 0; j < n-i-1; j++) { if (arr[j] > arr[j+1]) { // Swap elements int temp = arr[j]; arr[j] = arr[j+1]; arr[j+1] = temp; } } } }

2. Selection Sort

void selectionSort(int arr[], int n) { for (int i = 0; i < n-1; i++) { int minIdx = i; for (int j = i+1; j < n; j++) { if (arr[j] < arr[minIdx]) { minIdx = j; } } // Swap minimum element with first element int temp = arr[minIdx]; arr[minIdx] = arr[i]; arr[i] = temp; } }

3. Insertion Sort

void insertionSort(int arr[], int n) { for (int i = 1; i < n; i++) { int key = arr[i]; int j = i - 1; while (j >= 0 && arr[j] > key) { arr[j + 1] = arr[j]; j = j - 1; } arr[j + 1] = key; } }

10.2 Algoritma Searching

1. Linear Search

Linear Search memeriksa setiap elemen dalam array secara berurutan hingga menemukan elemen yang dicari.

int linearSearch(int arr[], int n, int target) { for (int i = 0; i < n; i++) { if (arr[i] == target) { return i; // Return index if found } } return -1; // Return -1 if not found }

2. Binary Search

Binary Search bekerja pada array yang sudah terurut dengan membagi array menjadi dua bagian secara berulang.

int binarySearch(int arr[], int low, int high, int target) { while (low <= high) { int mid = low + (high - low) / 2; if (arr[mid] == target) return mid; else if (arr[mid] < target) low = mid + 1; else high = mid - 1; } return -1; // Element not found }
